README.txt
==========

Build and run kuromoji-server as follows:

1. Run Maven using

    % mvn jetty:run


2. Browse to

    http://localhost:8080/kuromoji/
  
  type Japanese text and press RET


Note that the dot command from Graphviz (http://www.graphviz.org/) is required
for Viterbi visualisation.
